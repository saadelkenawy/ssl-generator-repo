🚀 Quick Install (2 steps)

# Step 1 — Add the repository
sudo curl -o /etc/yum.repos.d/ssl-generator.repo \
  https://YOUR_GITHUB_USERNAME.github.io/ssl-generator-repo/ssl-generator.repo

# Step 2 — Install
sudo dnf install ssl-generator      # RHEL 8+ / Rocky / AlmaLinux / Fedora
sudo yum install ssl-generator      # RHEL 7 / CentOS 7

⬇️ Direct RPM Download

sudo rpm -ivh https://YOUR_GITHUB_USERNAME.github.io/ssl-generator-repo/x86_64/ssl-generator-1.0.0-1.x86_64.rpm

Or download manually: ssl-generator-1.0.0-1.x86_64.rpm

💡 Usage

ssl-generator            # Interactive mode
ssl-generator --help     # Show help

Mode 1 — Full PKI generation: Root CA → Intermediate CA → Server Certificate
Mode 2 — Issue additional server certs from an existing CA directory

🗑️ Uninstall

sudo dnf remove ssl-generator
# or
sudo yum remove ssl-generator
